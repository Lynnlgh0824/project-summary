# 项目日志系统使用说明

## 📁 目录说明

```
project summary/
├── project-log.html           # 项目日志系统
├── 项目经验总结.html             # 项目经验总结文档
├── project-log-data.json       # 数据文件（首次保存时创建）
└── README.md                  # 本说明文件
```

---

## 💾 自动保存机制

### 工作原理

1. **首次使用**：
   - 点击 "➕ 添加日志" 保存第一条记录
   - 浏览器会弹出文件保存对话框
   - 选择保存位置（建议保存到本目录）
   - 文件名：`project-log-data-YYYY-MM-DD.json`

2. **后续使用**：
   - 每次添加/修改日志，**自动保存**到同一个文件
   - 无需手动操作，数据实时同步
   - 即使清除浏览器数据，只需重新打开文件即可恢复

---

## 🚀 快速开始

### 方式一：新建数据文件

1. 打开 `project-log.html`
2. 点击 "➕ 添加日志"
3. 填写内容并保存
4. 在弹出的保存对话框中，选择保存到本目录
5. ✅ 完成！后续所有操作都会自动保存

### 方式二：打开已有数据文件

1. 打开 `project-log.html`
2. 点击 "📂 打开数据文件"
3. 选择之前保存的 `project-log-data-*.json`
4. ✅ 数据已加载，后续操作会自动更新此文件

## ⚠️ 注意事项

1. **首次保存需要选择位置** - 浏览器安全限制，必须用户手动选择一次
2. **后续自动保存** - 选择了文件位置后，所有操作都会自动更新
3. **浏览器兼容性** - 需要使用 Chrome、Edge 等支持 File System Access API 的浏览器
4. **定期备份** - 建议定期复制 JSON 文件到其他位置作为备份

---

## 🔧 技术说明

- **File System Access API** - 允许网页直接读写本地文件
- **自动保存** - 每次保存日志时自动写入文件
- **无需服务器** - 纯静态页面，直接在浏览器中打开
- **数据安全** - 所有数据保存在本地，不上传到云端
